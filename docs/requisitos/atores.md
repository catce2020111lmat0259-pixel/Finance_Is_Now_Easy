# Atores
## Sistema

Descrição:
Responsável por processar dados, validar informações e armazenar registros no banco de dados.

## Usuário do sistema (autenticado)

Descrição:
Usuário responsável por utilizar o sistema para gerenciar suas finanças pessoais.

Permissões:
- Inserir receitas e despesas
- Editar e excluir lançamentos
- Classificar transações por categoria
- Visualizar relatórios e gráficos
- Criar, editar e acompanhar metas financeiras
- Visualizar dashboard financeiro
- Configurar preferências básicas do sistema

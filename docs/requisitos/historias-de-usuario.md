# Histórias de Usuário

## HU01 - Inserir receitas manualmente
Como usuário do sistema, quero inserir minhas receitas manualmente (ex.: salário, mesada), para acompanhar minha entrada de dinheiro.

## HU02 - Inserir despesas manualmente
Como usuário do sistema, quero inserir minhas despesas manualmente (ex.: alimentação, transporte, lazer), para controlar meus gastos.

## HU03 - Editar e excluir lançamentos
Como usuário do sistema, quero editar ou excluir lançamentos já registrados, para manter meus dados atualizados.

## HU04 - Classificação por categorias
Como usuário do sistema, quero classificar receitas e despesas por categorias, para organizar melhor minhas finanças.

## HU05 - Resumo de despesas por categoria
Como usuário do sistema, quero visualizar o resumo de despesas por categoria, para entender onde estou gastando mais.

## HU06 - Relatório mensal
Como usuário do sistema, quero visualizar um relatório mensal de receitas e despesas e exportar os dados, para analisá-los fora do sistema.

## HU07 - Gráficos por categoria
Como usuário do sistema, quero gráficos (pizza e/ou barras) mostrando gastos por categoria, para tornar a visualização e interpretação mais clara.

## HU08 - Comparativo entre meses
Como usuário do sistema, quero um comparativo entre meses (quanto gastou em janeiro, fevereiro, março, etc), para manter o controle mensal de gastos.

## HU09 - Metas financeiras
Como usuário do sistema, quero criar e personalizar metas financeiras, para organizar melhor meus objetivos e acompanhar o progresso em relação aos gastos.

## HU10 - Avisos de orçamento
Como usuário financeiro, quero receber avisos quando meus gastos ultrapassarem o limite de orçamento definido, para tomar decisões corretivas antes que o mês encerre.

## HU11 - Dashboard inicial
Como usuário, quero visualizar no dashboard inicial o saldo atual, o total de receitas e o total de despesas do período, para ter uma visão rápida da minha situação financeira.

## HU12 - Interface responsiva
Como usuário, quero navegar pela aplicação em qualquer dispositivo com uma interface clara e de fácil compreensão, para utilizar o sistema sem dificuldades independentemente da tela.

## HU13 - Filtros de pesquisa
Como usuário financeiro, quero filtrar minhas transações por categoria ou período, para que eu possa encontrar lançamentos específicos e analisar meus gastos de forma segmentada.

## HU14 - Personalização de cores
Como usuário do app, quero personalizar as cores da interface, para que eu possa adaptar a aparência do sistema às minhas preferências visuais.

## HU15 - Persistência de dados
Como usuário financeiro, quero que meus dados fiquem armazenados com segurança, para que eu não perca minhas informações ao fechar o aplicativo ou sair da sessão.

## HU16 - Login e autenticação
Como usuário do sistema, quero realizar login na aplicação, para acessar minhas informações de forma segura.
